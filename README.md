혼밥하지 마세요! 새로운 만남! '모디라' 를 이용하세요


## 프로젝트 개요 
<br>
- 밥을 먹고 싶은데, 혼자 먹기 싫고, 새로운 만남을 하고 싶을 때! 공짜로 밥을 먹고 싶을때! 밥친구 찾기 플렛폼 모디라 서비스 입니다! 
<br>
1  다양한 유형의 모임을 만들어보세요! 
<br>
2. 버튼으로 간편하게 참여신청을 보낼 수 있어요!
<br>
3. 채팅을 통해 약속을 잡아보세요!
<br>
4. 모임이 불안하신가요? 모임장의 하트점수를 확인해보세요!
